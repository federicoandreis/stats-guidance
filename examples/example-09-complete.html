27  Example 9: Staffing and Quality Outcomes

Correlation Analysis with Confounding Considerations

27.1 Scenario Overview

Method: Correlation analysis Complexity: Medium-High Time to complete: 3-4 hours QASQA questions: Q2 (Design); Q3 (Analysis)

27.2 Scenario

You are a CQC analyst investigating whether staffing levels are associated with quality outcomes in residential care homes. Recent media coverage has suggested that homes with lower staffing ratios have worse outcomes, but you need to understand the relationship properly before making policy recommendations.

Analytical Question: Is there a relationship between registered nurse staffing ratios and resident safety outcomes (pressure ulcer rates) in residential care homes? If so, how strong is it, and what other factors might explain the relationship?

Context: CQC collects data on staffing levels and safety outcomes from ~800 residential care homes. Stakeholders want evidence about whether staffing levels are associated with outcomes, to inform guidance and inspection focus. However, you know that other factors (home size, resident complexity, location) might confound this relationship.

Why correlation, not z-scoring?: This is a thematic analysis exploring relationships between variables, not comparing individual provider performance. The goal is to understand patterns across the sector, not flag specific homes.

27.3 Learning Objectives

By working through this example, you will learn to:

  1. Formulate clear research questions before analysing
  2. Visualise relationships using scatter plots with annotations
  3. Choose between Pearson and Spearman based on data characteristics
  4. Calculate and interpret correlation coefficients with confidence intervals
  5. Identify and handle outliers appropriately
  6. Consider confounding variables and their impact
  7. Conduct sensitivity analyses to test robustness
  8. Distinguish correlation from causation in reporting
  9. Communicate uncertainty and limitations clearly
  10. Document analytical decisions for QA

27.4 Dataset Description

Synthetic Data

This synthetic dataset represents 800 residential care homes with the following variables:

Variable Description Type Range
home_id Unique care home identifier Character CH001-CH800
region Geographic region Categorical North, Midlands, South, London
home_size Number of registered beds Discrete 10-120
nurse_ratio Registered nurses per 50 residents Continuous 0.5-8.0
pressure_ulcer_rate Pressure ulcers per 100 resident-months Continuous 0.2-12.5
resident_complexity Average dependency score (1-5 scale) Continuous 1.8-4.5
cqc_rating Overall CQC rating Ordinal 1-4 (Inadequate to Outstanding)
urban_rural Location type Categorical Urban, Rural

Data characteristics: - Complete data for all 800 homes (no missing values) - Realistic distributions based on CQC sector knowledge - Includes confounding variables (size, complexity, location) - Contains 3 planted outlier homes with unusual staffing or outcomes

The planted truth (visible in the generation code below, because knowing the right answer is what makes a simulation instructive): better staffing genuinely reduces pressure ulcers in this data – but resident complexity pushes both staffing and ulcer rates up. The confounder therefore partially masks the staffing effect: the correlation you will observe understates the relationship that was built in. Watch for it.

27.5 Step 1: Define Research Question

Primary question: Is nurse staffing ratio associated with pressure ulcer rates?

Hypothesised relationship: Higher nurse ratios should be associated with lower pressure ulcer rates.

Alternative explanations: - Reverse causation: Homes with higher ulcer rates might hire more nurses - Confounding by complexity: Complex residents → higher staffing AND higher ulcers - Confounding by size: Larger homes have different staffing models - Regional variation: Different regions have different norms

27.6 Step 2: Load and Explore Data

  • R
  • Python
set.seed(42)
n_homes <- 800

# Generate home characteristics
home_data <- data.frame(
  home_id = paste0("CH", sprintf("%03d", 1:n_homes)),
  region = sample(c("North", "Midlands", "South", "London"), n_homes, 
                  replace = TRUE, prob = c(0.3, 0.25, 0.3, 0.15)),
  home_size = sample(10:120, n_homes, replace = TRUE),
  urban_rural = sample(c("Urban", "Rural"), n_homes, replace = TRUE, prob = c(0.7, 0.3))
)

# Generate resident complexity
home_data$resident_complexity <- 2.5 + 
  (home_data$home_size / 100) * 0.8 + 
  ifelse(home_data$urban_rural == "Urban", 0.3, 0) +
  rnorm(n_homes, 0, 0.4)
home_data$resident_complexity <- pmin(pmax(home_data$resident_complexity, 1.8), 4.5)

# Generate nurse ratios
home_data$nurse_ratio <- 3.5 + 
  (home_data$resident_complexity - 2.5) * 1.2 +
  ifelse(home_data$region == "London", 0.8, 0) +
  ifelse(home_data$region == "North", -0.5, 0) +
  (home_data$home_size - 65) / 50 * 0.5 +
  rnorm(n_homes, 0, 1.2)
home_data$nurse_ratio <- pmin(pmax(home_data$nurse_ratio, 0.5), 8.0)

# Generate pressure ulcer rates (negatively correlated with staffing)
home_data$pressure_ulcer_rate <- 8 - 
  (home_data$nurse_ratio - 3.5) * 0.6 +
  (home_data$resident_complexity - 2.5) * 1.5 +
  rnorm(n_homes, 0, 1.8)
home_data$pressure_ulcer_rate <- pmin(pmax(home_data$pressure_ulcer_rate, 0.2), 12.5)

# Add 3 outliers
outlier_indices <- c(50, 250, 600)
home_data$nurse_ratio[outlier_indices[1]] <- 7.5
home_data$pressure_ulcer_rate[outlier_indices[1]] <- 3.0
home_data$nurse_ratio[outlier_indices[2]] <- 1.2
home_data$pressure_ulcer_rate[outlier_indices[2]] <- 11.0
home_data$nurse_ratio[outlier_indices[3]] <- 6.5
home_data$pressure_ulcer_rate[outlier_indices[3]] <- 10.5

# Generate CQC ratings
rating_score <- 2.5 - 
  (home_data$pressure_ulcer_rate - 6) / 4 +
  (home_data$nurse_ratio - 3.5) / 3 +
  rnorm(n_homes, 0, 0.6)
home_data$cqc_rating <- cut(rating_score, 
                             breaks = c(-Inf, 1.5, 2.5, 3.5, Inf),
                             labels = c(1, 2, 3, 4))
home_data$cqc_rating <- as.numeric(as.character(home_data$cqc_rating))

Results: Pearson’s r = −0.27 (95% CI −0.33 to −0.20, p < 0.001); Spearman’s ρ = −0.25, confirming the relationship is roughly linear and not an artefact of the outliers. A weak-to-moderate negative association: staffing shares only about 7% of its variation with ulcer rates (r² ≈ 0.07). Before reading that as “staffing barely matters”, recall the planted truth – complexity pushes staffing and ulcers in the same direction, so this marginal correlation understates the staffing effect that was built into the data. That is confounding working in the direction people rarely expect: it can hide real relationships as readily as manufacture spurious ones.

27.9 Step 5: Report Findings

Summary

Analysis of 800 residential care homes shows a weak-to-moderate negative correlation between nurse staffing ratios and pressure ulcer rates (Pearson’s r = −0.27, 95% CI −0.33 to −0.20, p < 0.001; Spearman’s ρ = −0.25). Homes with higher nurse-to-resident ratios tend to have lower pressure ulcer rates, but staffing shares only about 7% of its variation with outcomes – resident complexity, care practices, and environment clearly matter at least as much.

The headline number needs careful handling in both directions. It does not establish that staffing drives outcomes: reverse causation (homes with problems hiring more staff) and confounding remain live explanations. Nor does its modest size establish that staffing barely matters: resident complexity is associated with both higher staffing and higher ulcer rates, and a confounder that pushes the two variables the same way will shrink the observed correlation below the true relationship. Untangling that requires analysis that adjusts for complexity – multivariate work beyond correlation, for which we recommend Guild support.

Recommendations

  1. For policy: Staffing-outcome relationship warrants attention, but avoid simplistic staffing ratio targets
  2. For inspection: Consider staffing in context of resident complexity and other quality indicators
  3. For further investigation: Examine outlier homes (especially high-staffing/high-ulcer homes) to understand why staffing alone doesn’t guarantee good outcomes
  4. For research: Longitudinal study needed to establish causation; qualitative work to understand mechanisms

27.10 QA Documentation

Method justification: Pearson and Spearman both reported – the scatter looks linear and the outliers turn out not to steer the coefficient, so the two agree

Assumptions checked: - Linearity: confirmed via scatter plot, corroborated by Pearson-Spearman agreement - Outliers: three identified on the plot; sensitivity analysis below shows minimal influence - Independence: each home is one observation

Limitations: - Cross-sectional data (cannot infer causation or direction) - Confounding by resident complexity is known to be present and unadjusted – the marginal correlation likely understates the staffing relationship - Ecological analysis (home-level, not resident-level) - Staffing measured at one time point (does not capture variation)

Sensitivity analyses: - Excluding the three outliers: r moves from −0.267 to −0.262 (robust) - Spearman vs Pearson: −0.25 vs −0.27 (consistent; no outlier or non-linearity artefact)

Next steps: investigate CH600 (high staffing and high ulcers – the combination no simple story explains); multivariate analysis adjusting for complexity, with Guild support

27.11 Key Learning Points

  1. Always visualise first: the scatter plot revealed the outliers and the overall pattern
  2. Context matters: correlation strength depends on field – in complex health and social care data, an r of −0.27 on a safety outcome is worth attention
  3. Confounding cuts both ways: complexity affects both staffing and outcomes, and here it masks the staffing relationship rather than manufacturing it
  4. Correlation ≠ causation: multiple explanations remain live
  5. Report completely: coefficient, confidence interval, p-value, scatter plot, limitations
  6. Sensitivity analysis: test robustness to outliers and to the choice of coefficient
  7. Know when to escalate: only adjusted, multivariate analysis can separate staffing from complexity

This example demonstrates a complete correlation analysis workflow suitable for CQC thematic work, with appropriate attention to confounding, outliers, and limitations.

pearson_all <- cor.test(home_data$nurse_ratio, 
                        home_data$pressure_ulcer_rate,
                        method = "pearson")

spearman_all <- cor.test(home_data$nurse_ratio, 
                         home_data$pressure_ulcer_rate,

correlation_results <- data.frame(
  Method = c("Pearson", "Spearman"),
  n = c(nrow(home_data), nrow(home_data)),
  Coefficient = c(pearson_all$estimate, spearman_all$estimate),
  CI_Lower = c(pearson_all$conf.int[1], NA),
  CI_Upper = c(pearson_all$conf.int[2], NA),
  P_Value = c(pearson_all$p.value, spearman_all$p.value)
)

## Python

```{python}
#| label: correlation-py
#| echo: true

pearson_r, pearson_p = stats.pearsonr(home_data_py['nurse_ratio'], 
                                       home_data_py['pressure_ulcer_rate'])

spearman_rho, spearman_p = stats.spearmanr(home_data_py['nurse_ratio'], 
                                            home_data_py['pressure_ulcer_rate'])

n = len(home_data_py)
z = np.arctanh(pearson_r)
se = 1 / np.sqrt(n - 3)
ci_lower = np.tanh(z - 1.96 * se)
ci_upper = np.tanh(z + 1.96 * se)

results_df = pd.DataFrame({
    'Method': ["Pearson", "Spearman"],
    'n': [n, n],
    'Coefficient': [pearson_r, spearman_rho],
    'CI_Lower': [ci_lower, np.nan],
    'CI_Upper': [ci_upper, np.nan],
    'P_Value': [pearson_p, spearman_p]
})

results_df['Coefficient'] = results_df['Coefficient'].round(3)
results_df['CI_Lower'] = results_df['CI_Lower'].round(3)
results_df['CI_Upper'] = results_df['CI_Upper'].round(3)

print("\nCorrelation Results")
print(results_df.to_string(index=False))
```

:::

**Results**: Pearson's r = −0.27 (95% CI −0.33 to −0.20, p < 0.001); Spearman's ρ = −0.25, confirming the relationship is roughly linear and not an artefact of the outliers. A weak-to-moderate negative association: staffing shares only about 7% of its variation with ulcer rates (r² ≈ 0.07). Before reading that as "staffing barely matters", recall the planted truth – complexity pushes staffing and ulcers in the *same* direction, so this marginal correlation *understates* the staffing effect that was built into the data. That is confounding working in the direction people rarely expect: it can hide real relationships as readily as manufacture spurious ones.

## Step 5: Report Findings

### Summary

> Analysis of 800 residential care homes shows a weak-to-moderate negative correlation between nurse staffing ratios and pressure ulcer rates (Pearson's r = −0.27, 95% CI −0.33 to −0.20, p < 0.001; Spearman's ρ = −0.25). Homes with higher nurse-to-resident ratios tend to have lower pressure ulcer rates, but staffing shares only about 7% of its variation with outcomes – resident complexity, care practices, and environment clearly matter at least as much.
>
> The headline number needs careful handling in both directions. It does not establish that staffing drives outcomes: reverse causation (homes with problems hiring more staff) and confounding remain live explanations. Nor does its modest size establish that staffing barely matters: resident complexity is associated with *both* higher staffing and higher ulcer rates, and a confounder that pushes the two variables the same way will shrink the observed correlation below the true relationship. Untangling that requires analysis that adjusts for complexity – multivariate work beyond correlation, for which we recommend Guild support.

### Recommendations

1. **For policy**: Staffing-outcome relationship warrants attention, but avoid simplistic staffing ratio targets
2. **For inspection**: Consider staffing in context of resident complexity and other quality indicators
3. **For further investigation**: Examine outlier homes (especially high-staffing/high-ulcer homes) to understand why staffing alone doesn't guarantee good outcomes
4. **For research**: Longitudinal study needed to establish causation; qualitative work to understand mechanisms

## QA Documentation

**Method justification**: Pearson and Spearman both reported – the scatter looks linear and the outliers turn out not to steer the coefficient, so the two agree

**Assumptions checked**:
- Linearity: confirmed via scatter plot, corroborated by Pearson-Spearman agreement
- Outliers: three identified on the plot; sensitivity analysis below shows minimal influence
- Independence: each home is one observation

**Limitations**:
- Cross-sectional data (cannot infer causation or direction)
- Confounding by resident complexity is known to be present and *unadjusted* – the marginal correlation likely understates the staffing relationship
- Ecological analysis (home-level, not resident-level)
- Staffing measured at one time point (does not capture variation)

**Sensitivity analyses**:
- Excluding the three outliers: r moves from −0.267 to −0.262 (robust)
- Spearman vs Pearson: −0.25 vs −0.27 (consistent; no outlier or non-linearity artefact)

**Next steps**: investigate CH600 (high staffing *and* high ulcers – the combination no simple story explains); multivariate analysis adjusting for complexity, with Guild support

## Key Learning Points

1. **Always visualise first**: the scatter plot revealed the outliers and the overall pattern
2. **Context matters**: correlation strength depends on field – in complex health and social care data, an r of −0.27 on a safety outcome is worth attention
3. **Confounding cuts both ways**: complexity affects both staffing and outcomes, and here it *masks* the staffing relationship rather than manufacturing it
4. **Correlation ≠ causation**: multiple explanations remain live
5. **Report completely**: coefficient, confidence interval, p-value, scatter plot, limitations
6. **Sensitivity analysis**: test robustness to outliers and to the choice of coefficient
7. **Know when to escalate**: only adjusted, multivariate analysis can separate staffing from complexity
